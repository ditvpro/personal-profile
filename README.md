# Personal Profile Website

A modern personal profile website built with vanilla HTML, CSS, and JavaScript.

## Features

- **Hero Section** — Introduction with avatar, tagline, and CTA buttons
- **Portfolio Section** — Project showcase with interactive cards
- **Blog Section** — Blog posts displayed as cards
- **Dark Mode** — Toggle with localStorage persistence and system preference detection
- **Scroll Animations** — Smooth fade-in effects using IntersectionObserver
- **Responsive Design** — Mobile-first layout that works on all screen sizes

## Tech Stack

- HTML5
- CSS3 (Custom Properties, Grid, Flexbox)
- Vanilla JavaScript (no frameworks)
- Google Fonts (Inter)

## Project Structure

```
├── index.html
├── css/
│   └── style.css
└── js/
    └── main.js
```

## Getting Started

Open `index.html` in your browser, or serve it locally:

```bash
npx serve .
```
